Introduction
===================



Scientific Computing 是 Coursera.org 上一门课程（`Scientific Computing <https://class.coursera.org/scientificcomp-004/class>`_），这门课程中，我们讲会学会科学计算的基础知识，并且可以使用 Matlab 解决具体的问题。


## 背景知识

* 线性代数
* 微分方程
* 编程基础：多少要懂点，比如循环的概念

## 所需的工具

* 计算机 + Matlab/Octave

Matlab 是首选。Octave 是一个免费的替代品。


## 课程中将会学到

* 两类问题
    1.  Initial Value Problem
    2.  Boundary Problem

   .

* 三种方法
    1.  Finite Differences (based on Taylor expansion)
    2.  Spetral Methods (Fourier transform)
    3. Finite Element (to solve problems with complex geometry)
